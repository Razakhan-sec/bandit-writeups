# Bandit Writeups

Personal writeups documenting my progress through [OverTheWire's Bandit wargame](https://overthewire.org/wargames/bandit/) — a beginner-friendly series of Linux and security fundamentals challenges.

## Objective

These are **not solution guides**. Each writeup breaks down the theory behind the commands and concepts needed for that level, along with screenshots at key checkpoints — but stops short of giving away the actual solve. The goal is to help readers (and future me) build the thinking process to solve each level independently, not just copy-paste an answer.

Expect:
- Command theory explained in plain language
- Checkpoint screenshots showing what you *should* see at each stage
- Common mistakes / warnings I ran into along the way
- No direct spoilers or final answers
